## üéì Learning Paths

### Path 1: Beginner (API Testing Only)
1. Read [README.md](README.md) - Quick Start section
2. Follow [SIMPLE_INSTALLATION_GUIDE.md](SIMPLE_INSTALLATION_GUIDE.md)
3. Try [examples/complete_showcase.feature](examples/complete_showcase.feature) with `@http` tag
4. Read [judo-steps-reference-en.md](judo-steps-reference-en.md) for available steps

### Path 2: Intermediate (API + Screenshots)
1. Complete Beginner path
2. Read [docs/SCREENSHOT_INTEGRATION.md](docs/SCREENSHOT_INTEGRATION.md)
3. Follow [examples/complete_project_setup.md](examples/complete_project_setup.md)
4. Implement custom runner from examples

### Path 3: Advanced (API + UI + Screenshots)
1. Complete Intermediate path
2. Read [.kiro/migration-guide-playwright.md](.kiro/migration-guide-playwright.md)
3. Study [examples/environment_playwright.py](examples/environment_playwright.py)
4. Try [examples/playwright_integration.feature](examples/playwright_integration.feature)
5. Build hybrid tests (API + UI)

### Path 4: Developer/Contributor
1. Complete Advanced path
2. Read [.kiro/JUDO_FRAMEWORK_GUIDE.md](.kiro/JUDO_FRAMEWORK_GUIDE.md)

[1.5.9.5] - 2026-01-08
-----------------------
MEJORAS FINALES: Footer Solo Logo Completamente Implementado

- Footer Solo Logo: Footer muestra únicamente el logo cuando se configura 
  show_creator: false y show_logo: true
- Sin Texto Adicional: Eliminado completamente el texto "Framework creado por..."
- Logo Centrado: Logo del footer centrado con diseño limpio
- Configuración Intuitiva: Configuración simple en archivo JSON
- Soporte Base64 Completo: Funciona perfectamente con logos en formato base64

Configuración Final:
{
  "footer": {
    "show_creator": false,
    "show_logo": true
  },
  "branding": {
    "primary_logo": "data:image/png;base64,TU_LOGO_BASE64_AQUI"
  }
}

Archivos Actualizados:
- judo/reporting/html_reporter.py: Método _generate_footer() optimizado
- test_footer_solo_logo.py: Test completo validando funcionalidad
- ejemplo_configuracion_logo.json: Configuración actualizada

NUEVA FUNCIONALIDAD: Reportes HTML Completamente Personalizables

- Logos Personalizables: Soporte para logos principales, secundarios y de empresa

[1.5.9.1] - 2026-01-07
-----------------------
MAJOR FEATURE: Validación Completa de Contratos de Servicios

- Validación OpenAPI: Carga y validación completa contra especificaciones OpenAPI 3.0
- Validación AsyncAPI: Soporte para especificaciones AsyncAPI
- Validación de Esquemas: Validación contra esquemas específicos por nombre
- Validación de Tipos: Validación avanzada de tipos de datos
- Validadores de Formatos Avanzados: Email, URL, UUID, Fecha ISO, Teléfono
- 20+ Nuevos Pasos de Validación en inglés y español

[1.5.9.0] - 2026-01-07
-----------------------
MAJOR FEATURE: Configuración Centralizada via .env

- Configuración Centralizada: Migración completa de configuraciones del runner al .env
- Runners Ultra-Simples: Solo necesitan heredar de BaseRunner()
- Variables de Entorno Soportadas: 15+ variables para configuración completa
- Carga Automática de .env: Busca automáticamente archivos .env
- Compatibilidad hacia atrás: Cero breaking changes

Variables de Entorno Principales:
- JUDO_FEATURES_DIR: Directorio con archivos .feature
- JUDO_OUTPUT_DIR: Directorio para reportes
- JUDO_PARALLEL: Ejecutar en paralelo
- JUDO_MAX_WORKERS: Número máximo de hilos
- JUDO_CONSOLE_FORMAT: Formato de consola
- JUDO_TIMEOUT: Timeout en segundos
